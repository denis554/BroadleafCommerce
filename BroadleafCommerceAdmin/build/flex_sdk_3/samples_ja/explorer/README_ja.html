Explorer の実行

Component Explorer アプリケーションは、Adobe® Flex™ 2 フレームワークで使用可能な MXML コンポーネントの大半について簡単な使用法を示します。Adobe Flex Charting 2 をインストールすると、コンポーネントの使用例も参照できます。

既存の Component Explorer アプリケーションへのチャートの追加

  1. Flex Charting のインストール手順に従って、Flex Component Explorer をインストールして更新します。
  2. コマンドプロンプトまたはファイルエクスプローラで、"flex_sdk_dir/samples/explorer/charts" に移動します。
  3. "build.bat" バッチファイル (Windows) または "build.sh" シェルスクリプト (Mac/UNIX) を実行して、サンプルをコンパイルします。

Adobe® Flex™ Builder™ 2 に関する注意事項

Component Explorer のサンプルコードを Flex Builder 内部から編集する場合でも、コマンドラインコンパイラを使用して個々のサンプルをコンパイルする必要があります。Flex Builder はソースフォルダのルートの下にあるアプリケーションをコンパイルし実行します。

Adobe® Flex™ SDK に関する注意事項

Component Explorer のビルドスクリプトは、ローカルファイルシステム (ブラウザでは file:///) からアプリケーションが実行されることを前提としています。Web サーバー (ブラウザでは http://) からアプリケーションを実行する場合は、ビルドスクリプトの -use-network 設定を true に変更してください。
